What is Steam Discovery ?

Steam Discovery is an IMAC (engineering school) student web project.
This web app helps you to discover new games directly from Steam by giving you 4 random titles !
Each game contains a lot of informations to guide you to choose one more than another.

Wow, but how does it work ?

It is quite easy, you are just a few clicks away ! Just visit our main page here to load 4 random games. If they don't please to you, Then reroll them :)

To what extent does it deals with Steam ? Is it a service delivered by Steam ?

In order to provide you this little web app, we used Steam's API, but Steam Discovery is not affiliated with Valve, Steam, or any of their partners.

What did you use to make this application ?

Steam Web API available here, exploited with PHP, some pure AJAX magic. The website is made with HTML5 & CSS3 to bring you eye-popping animations ! Oh did we forgot to mention jQuery ?
